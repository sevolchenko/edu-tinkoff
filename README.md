# Чат-бот в Telegram  [@linkmonitoringbot](https://t.me/linkmonitoringbot)

## Для отслеживания контента по ссылкам
### Поддерживаемые типы ссылок:
- GitHub
- StackOverflow

## Технологии
- Java 17
- Spring Boot 3.0.1
- Maven
- [Java Telegram Bot API](https://github.com/pengrad/java-telegram-bot-api)
- PostgreSQL
- RabbitMQ
- GitHub Workflows (CI)
- CheckStyle
- Prometheus
- Grafana
